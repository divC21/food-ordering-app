# HungryMe 
HungryMe is a food delivery application that allows users to order food from their favorite restaurants and have it delivered to their doorstep. The application provides a user-friendly interface for browsing menus, placing orders.

## Features
- Browse restaurants 
- View menus and prices
- Add items to cart
- Track order status (TBD)
- Contact support
- Grocery module (TBD)

## React concepts used
- Functional components
- Hooks (useState, useEffect)
- Context API 
- React Router
- Fetch for API calls
- Tailwind CSS for styling
- Redux for state management
- Unit testing with Jest and React Testing Library
- Parcel for bundling
- Custom hooks for reusable logic
